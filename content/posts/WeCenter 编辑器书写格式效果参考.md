**`**加粗**`：单位常常要忙很多事的时候，领导便会皱着眉头，语重心长的对我们说，你们都是搞文字工作的，不多干点实际工作，怎么能有创作的源泉呢？**

*`*斜体*`：有一次我忍不住问他：“意思是生活，是一泓清泉，这泓清泉可以浇灌在每个人身上浇灌出无数的灵感？”*

按一个次`Enter` 键正常换行:领导拍**着大腿说**，对了，对了，就是这个意思。你看，你这样的总结，虽然没有我那个总结漂亮，但实在啊，为什么实在，就是你长期接触基层工作嘛。
按两次 `Enter` 键即留一空行为下一段落：听了他的话，我紧紧的咬紧了下嘴唇，拼命的点头，表示拥护他英明的观点。哈哈哈换回哈哈换回哈换回好哈哈哈换回

> 引用：不过说实在的，郁闷归郁闷，结合自己的创作历程看，领导其实说的也没有错，我的大多数文字均来自现实生活的体验，别人的，自己的，我一直认为，只要每天经历在，创作的源泉便会不断的涌现。所以，我得承认，生活是创作的一泓清泉，这当然包括大量的从事基层的繁杂工作。
但这个观点到近半年的时间，却不得不让我开始一点点怀疑，真的仅仅是一泓清泉？

###h3代码片段,支持两种代码嵌套

以下为默认的 ```{ { {your code } } }``` 嵌套代码，这个是块级代码。(这里防止格式生效给每个大括号间加了个空格)

{{{.aw-custom-page{padding: 20px;background: #fff;}

.aw-custom-page h2{font-size: 22px}

.aw-custom-page h4{font-size: 16px;}
.aw-custom-page h5{font-size: 14px;}
.aw-custom-page h6{font-size: 12px;}
.aw-custom-page p{line-height: 24px;}
.aw-custom-page ul, .aw-custom-page ol{padding:0 0 0 20px;}
.aw-custom-page ul{list-style-type: square;}

.aw-custom-page img{margin:10px 0;}}}

###以下为正常段落
到了下午，小公主四点半放学，于是我不得不从城市的一头，坐车到另一头接她，然后是买菜做饭，忙得心慌意乱的。等到吃过晚饭，不久我又不得不陪她早在上床，哄她入睡，等到她妈妈终于回家，她已经入睡，这时我爬起来，心里想，终于可以有点自己的时间了吧，于是打开电脑想写点东西，结果满脑子都是白天忙碌留下来的各种声音：买菜时我们匆忙的脚步声、我在厨房做饭小公主在客厅哭笑的声音；我准备些东西老婆在客厅厨房卧室弄出的噼里啪啦的声音------

这时我无法选择，只有无奈的关掉电脑，瘫坐在沙发上，一个字都不想写下去。但即便没有这些声音，要在一天一堆琐事里挑选一些有价值的东西写下去，也是极其困难的，因为真的无话可写。于是这一天就这样放弃了。

一天天如此，这样荒废的日子，我数着，已经过了33天，但没有尽头。因为老婆没有周末，没有法定节假日，所以，我只能准备很多年一个人照顾小公主的生活，我很悲观的想，我的创作生涯，难道就要这样结束。

生活倒是有了，填得满满的，但我发现，灵感已经没有了。也就是说，生活这泓清泉已经浇灌不出我的灵感了。所以现在我怀疑，生活并非都是一泓清泉，很多时候，它有可能还是一片片沙漠，一片片的覆盖下来，就掩盖了喘息着想写东西的人。
