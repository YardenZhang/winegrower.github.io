从去年开始就用了阿里百秀的这个模板，因为和WineShe社区的模板结合地很好，就一直延续到现在，但当时用的XIU 4.1版本，PC端还可以，移动端显示很不友好。之前一直要升级模板，但因为模板部分内容已经有修改，所以等到现在。

昨天有时间，将之前个人博客修改的代码重新找了一遍，这样就准备将个人博客模板更新到5.3。


<!--more-->


从4.1到5.3，我的模板修改的内容如下：

1、header文件中的登陆和注册 按钮提到 logo下面：

    <?php if( _hui(‘sign_f’) ){
    if( is_user_logged_in() ){
    global $current_user;
    get_currentuserinfo();
    ?>
    <div class=”user-welcome”>
    <a title=”点此进入后台” href=”<?php echo esc_url(‘/blog/wp-admin/’) ?>”><?php echo hui_get_avatar($user_id=$current_user->ID, $user_email=$current_user->user_email, $src=true); ?></a>
    <strong><?php echo $current_user->display_name ?></strong><span class=”text-muted”><?php echo __(‘欢迎登录！’, ‘haoui’) ?></span>
    </div>
    <p class=”user-logout”>
    <a href=”<?php echo wp_logout_url() ?>”><?php echo __(‘退出’, ‘haoui’) ?></a>
    </p>
    <?php }else{ ?>
    <div class=”user-signin”>
    <a class=”login btn btn-normal btn-primary” href=”http://www.wineshe.com/account/login/”><?php echo __(‘登陆’, ‘haoui’) ?></a>
    <a class=”register btn btn-normal btn-success” href=”http://www.wineshe.com/account/register/”> <?php echo __(‘注册’, ‘haoui’) ?></a>
    </div>
    <?php } ?>
    <?php } ?>
    <?php hui_nav_menu(); ?>
